Air Quality Index Forecast System

Project Overview

This project predicts air quality indicators using Machine Learning techniques. The dataset was cleaned, analyzed, visualized, and used to train a Random Forest Regression model.

Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

Project Workflow

1. Data Cleaning
2. Missing Value Handling
3. Data Visualization
4. Correlation Analysis
5. Feature Importance Analysis
6. Model Training
7. Prediction and Evaluation

Model Performance

- R² Score: 0.646
- Mean Absolute Error (MAE): 22.40

- Dataset Preview
- Dataset Information
- Correlation Heatmap
- Feature Importance Graph
- Actual vs Predicted Graph
- Model Performance Metrics
